# Usage
<!-- usage -->
```sh-session
$ npm install -g @alkamin/indigo
$ indigo COMMAND
running command...
$ indigo (-v|--version|version)
@alkamin/indigo/1.1.2 linux-x64 node-v12.20.1
$ indigo --help [COMMAND]
USAGE
  $ indigo COMMAND
...
```

## `indigo help [COMMAND]`

display help for indigo

```
USAGE
  $ indigo help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```

_See code: [@oclif/plugin-help](https://github.com/oclif/plugin-help/blob/v3.2.2/src/commands/help.ts)_

## `indigo new`

Scaffold a new library

```
USAGE
  $ indigo new

OPTIONS
  -h, --help  show CLI help
```

_See code: [src/commands/new/index.ts](https://github.com/alkamin/indigo/blob/v1.1.2/src/commands/new/index.ts)_
